## 介绍

Apache Shiro 是一个弹性的开源框架，可以处理认证、授权、企业级会话管理和加密。

Shiro 的首要目标是易于理解和使用。一个系统的安全性有时候十分复杂，甚至痛苦，但又是必不可少的。框架需要屏蔽复杂性并提供清晰直观的 API 以简化开发。
