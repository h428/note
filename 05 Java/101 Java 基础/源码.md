**源码阅读规则**

- 跑不起来不读：可以 Debug 才能阅读，否则事倍功半
- 解决问题就好，目的性
- 一条线所到底，选定和目的相关的一条路
- 无关细节略过
- 一般不读静态
- 一般动态读法
- 读源码先读骨架
