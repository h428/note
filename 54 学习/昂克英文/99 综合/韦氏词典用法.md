

# 1. 什么是韦氏词典

- 美国人 Native Speaker 他们用的词典，就像我们的新华字典一样
- 单词音标的标注方法不一样，都是直接使用 26 个字母（可能会在上方加一些横线或者点来区分字母音、重读、轻读）
- 例如，/tʃ/ 和 /dʒ/ 在韦氏中记为 /ch/ 和 /j/
- 就好像中文的这种标注方式一样：叒（若）叕（叕）


# 2. 为什么要用韦氏词典

- 很多词典（有道、欧陆）自带的美式 KK 音标会标错，有些会直接赋值 IPA 的发音过来，导致歧义
- 拿捏不准的情况下，可以参考韦词典的读法
- 以 financial 为例
