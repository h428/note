

# 1. 视频

## 1.1 粗剪

- `I, O` : **标记入点、出点**，可以直接将标记区域拖到序列中，或制作子剪辑
- `上/下` : 片段切换
- `左/右` : 逐帧跳转，按住 `shift` 可以跳转 5 帧
- `home/end` : 跳转到首尾
- `\` : 最大化序列中的时间线，用于快速预览
- `-, +` : 缩小、方法所在位置时间线，配合 `\` 使用
- `J, K, L` : **快退、停止、快进**
- `ctrl+shift+space` : **播放入点到出点**
- `ctrl+P` : 从当前位置播放到出点
- `ctrl+U` : 制作子剪辑，要求已经标记入点和出点，并选择保存目录
- `,` : 插入源到序列中的光标处
- `.` : 覆盖源到序列中的光标处，会覆盖等长度的内容，一般建议在上层轨道进行以保留原轨道
- `拖动鼠标时按住 shift` : 具有吸附功能
- `鼠标拖动插入片段时按住 ctrl` : 可在片段间插入而不覆盖原有内容，类似 `,`
- 若是拖动时不按 `ctrl` 默认会覆盖原有片段
- `V` : 移动工具
- `A` : 向前选择
- `shift+A` : 向后选择
- `X` : 标记当前光标所在片段（入点、出点）
- `ctrl+shift+x` : 取消标记的入点、出点
- `ctrl+alt+移动片段` : 进行片段的快速对调，若不按 `ctrl+alt` 直接移动会进行覆盖
- `delete/backspace` : 删除选中片段，会留下间隙，可以手动波纹删除
- `alt+backspace` : 删除片段以及对应间隙（后面的片段会自动前移）
- `C` : 剃刀工具，将当前片段划分为多个片段，并可以通过拉动边界补全回复
    - 注意剃刀工具将被分隔当前帧被划分给下一个片段，因此对帧的跳变处进行分割时，需要对跳变后的帧进行分割，否则会留一帧到后面的新片段
- 在序列中标记好入点、出点后，可以 :
    - `;` : 提升，即删除标记区域，但会保留间隙
    - `'` : 提取，删除标记区域以及对应的间隙，后面的内容会自动前移
    - 注意提升、提取都只对未锁定的轨道有效
    - 此外，由于提取会删除间隙，会影响所有未锁定的轨道，此时可以选择切换同步锁定来只对当前轨道生效而不删除其他轨道对应的内容
    - 提升、提取操作后，会自动分割片段、可随时滑动边界进行恢复
- 覆盖操作一般放置在上层轨道，以保留原轨道内容


# 1.2 精剪

- 可以直接调整片段边界，然后波纹删除间隙，以删除片头片尾
- 当多个片段连在一起时，只能减小边界，而不能扩充边界，此时，可以利用波纹编辑工具和滚动编辑工具进行调整 :
    - `B` : 波纹编辑工具，滑动边界的同时自动补充或者删除间隙
    - `N` : 滚动编辑工具，滑动边界的同时自动补充或者删除相邻片段在原视频中的对应长度
    - 移动工具、波纹编辑工具、滚动编辑工具都可以单击边界，滑动鼠标，以调整片段大小，只不过三者对于间隙的处理不同，还有相邻的片段，移动工具不能扩充，只能减少片段长度
    - 三者都可以在单击边界后，按住 `ctrl+左/右` 进行逐帧微调，同时按住 `shift` 可以调整 5 帧
    - 此外，还可以在单击边界后，定位好光标，按 `E` 直接裁剪边界到光标区域（相当于鼠标从边界拉到光标位置），一般用于多扩充了内容进行精细裁剪
- 滑动工具用于滑动片段中的视频内容 :
    - `Y` : 外滑工具，用于不改变片段长度的前提下，滑动该片段在原视频中的内容（相当于修改了在原视频的出入点后拉到序列中，长度不变）
    - `U` : 外滑工具，不改变当前片段长度，滑动当前片段两旁边的片段（一个会变长，一个会变短）
- 使用源覆盖序列中的片段（只支持覆盖） :
    - 首先在源中标记入点（必须是入点）
    - 然后按住 `alt`，并拖动源到对应片段，即可完成自动替换，长度即为片段的长度
    - 常用于同一场景的多镜头替换（必须先切割片段）
- 利用源插入/覆盖序列中的入点到出点区域 :
    - 首先在序列中标记入点、出点
    - 在源中标记入点或出点（只需标记一个）
    - 点击插入/覆盖，则会自动根据入点/出点进行插入/覆盖，插入/覆盖的长度即为在序列中标记区域的长度，当然可以利用精剪技巧重新调整长度
    - 该技巧一般用于快速覆盖指定区域，比覆盖片段更加强大，同时支持添加和覆盖，但操作较为复杂，可以直接对区域进行分割，然后直接覆盖片段
- 替换后，可以结合內滑、外滑调整位置，结合编辑工具调整长度

# 2. 音频

- 比较合适的音频分贝一般在 `-6 ~ -12 dB` 之间，超过 `0 dB` 一般认为爆音
- 有多个地方可以调整音频的音量，主要是 : 源文件或源剪辑的音频增益、序列中音频片段的音频增益、音频剪辑混合器、混音器
- 其中，可以把音频增益看做音频的输入，而音频剪辑混合器（音量控制线）、混音器看做输出音频的输出，最终的音量由输入和输出共同决定
    - 其中，音频剪辑混合器（音量控制线）、混音器在输入的基础上对音量进行调整
    - 注意，如果在音频增益中将音量跳得很低，那么在音量控制线中无论怎么调整也不会增大音量，因为输入源本身音量就很低
- `G` : 音频增益调整，可以统一调整源素材的音频增益，但更常见的是直接调整序列片段的音频增益，音频增益的体现就是音量，下文直接用音量代替
- `[, ]` : 通过音量控制线调整当前片段音量 -1 dB, +1 dB，按住 `shift` 可以 -3 dB, +3 dB
- 关键帧音量调整 : 按住 `ctrl` 可以在音频线上加点，然后利用添加的点调整音量，按住 `shift` 选择对应点后，可按 `[, ]` 进行调整或鼠标调整
- 关键帧音量调整可用于动态调整有人物对白时降低 BGM，无对白时正常或放大 BGM等
- 也可以利用钢笔工具加点，然后动态调整音量，可用于多片段的音量过渡
- 注意若要看到音量控制线（可能就是音高线？），必须开启 : 右击 fx -> 音量 -> 级别
- 按 `ctrl` 可以为点添加曲线，用于平滑过渡，可进行基础调音
- 默认情况下，启用的是剪辑关键帧，即单独调整每个片段的音高，可以在左侧设置模式为轨道关键帧，从而可以对整个轨道统一调整
- 平滑效果 : 片段和片段之间可以添加平滑效果
    - 效果 -> 音频过渡 -> 交叉淡化 -> 恒定功率，然后拖动到片段交界处
    - 可以选择多个片段，然后按 `ctrl+shift+D` 为所有交界处添加交叉淡化效果
    - 可以单独调整一个交叉淡化效果的长短
- 音频、视频同步问题 :
    - 默认情况，音频、视频不同步的地方，系统会在对应片段的左上角给出提示
    - 可以按住 `alt` 单独选中音频或视频，然后进行同步
    - 也可以右击左上角的提示，然后选择移动到同步/滑动到同步，也可以进行对齐
    - 对于取消了链接的视频和音频，编辑 -> 首选项 -> 时间轴，开启已经取消链接的视频的音频和视频的不同步提示（视频为 CC 2018 有，我的 CC 2014 没有）
    - 不建议取消视频音频之间的链接
- 序列中片段的音频的设置，其实就是音频剪辑混合器中的设置，二者保持同步，调整的是光标所在片段；而整个音轨的音频设置，则和
- 不同轨道的音频，则通过混音器进行控制最终的输出，其有多种音频控制模式（混音器都是针对整个轨道进行的） :
    - 写入 : 通过滑块自由写入音高线
    - 触动 : 可以自由写入，但松开滑块会自动回弹，比较高级的模式
    - 闭锁 : 不会回弹，类似写入，暂不知和写入有啥区别，好像
    - 读取 : 读取音高线，使用其控制音量
    - 关 : 忽略该轨道音高线的作用，这样即和输入保持一致
- 可以添加子混合轨道，然后在混音台中将多个轨道输出到子混合，以统一控制多个子混合轨道
- 最终，需要输出的轨道需要统一输出到主声道

- 写入会生成很多点，可以只用钢笔工具框选，然后 `del` 这些点
- 录音相关设置 :
    - 编辑 -> 首选项 -> 音频硬件，先设置好麦克风
    - 然后右击轨道图标的空白处，将画外音录制和启用轨道以进行录制拖动到图标栏中
    - 激活启用轨道已进行录制，然后单击画外音录制，即可开始录制音频

# 3. 图片

- 视频其实可以看做多个图片，因此实际上对图片的操作也可以应用于视频
- 设置图片默认时间 : 编辑 -> 首选项 -> 时间轴里进行设置 (CC 2014 没有找到)
- 对于图片，通过效果控件进行调整，在光标定位到指定的图片之后，点击效果控件中的运动，即可对图像进行 : 位置、缩放、旋转、透明度等调整，调整后可以在右侧点击重置按钮进行重置
- 锚点 : 旋转中心，可以移动（2014 CC 不可）
- 对于较大或较小的图片，想正好匹配当前序列，可以在序列中右击图片，然后选择缩放为帧大小或者设置为帧大小
    - 注意一般会优先选择设定为帧大小，因为设定为帧大小是通过缩放来实现的，而缩放为帧大小是直接设置宽高（此时其在效果控件的缩放指数是 100 %），会丢失高分辨的素材精度
    - 并且，设定为帧大小后，仍然可以通过重置按钮恢复原有大小
- 此外，对于较小的图片又不想缩放的，下层的视频剩余部分可能会被显示出来，此时可以设置一个颜色遮罩（如黑色）来屏蔽下层内容，让其只显示本层较小的图片
- 当多张连续图片总时长和音频时长不匹配时，可以根据音频时长，计算每张图片的平均时长，然后选取所有图片 -> 右击 -> 速度/持续时间 -> 设置持续时间并勾选波纹编辑，即可快速完成图片时长和音频时长的匹配
- 添加动画 :
    - 首先选定图片，打开效果控件，光标定位到起始帧位置，点击想设置的起始项，并设置初始的相关参数（例如缩放）
    - 然后定位到结尾帧位置，点击想设置的结束项，并设置结尾相关参数（例如缩放），然后就会自动过渡
    - 注意终点帧是按上/下键跳转后的前一帧，上下键默认跳转到片段起点，此外起点、终点要成对出现，否则可能效果错乱
    - 对具体帧的调节也可以直接通过鼠标进行，比如位置调整通过鼠标比较方便
    - 动画速度通过起始帧、结尾帧的距离进行控制
- 在效果中搜索裁剪，并添加到视频片段，即可为片段添加裁剪控件，用于的裁剪视频，并配合动画制作同时显示多角度的视频
- 效果面板中还有许多很常用的效果，可将效果添加到视频片段，即可在效果控件中出现对应的内容，并可设置详细的参数
- 其中，用于转场的过渡效果很常用，其分为音频过渡和视频过渡
- `ctrl+d` : 添加默认的视频转场
- `shift+D` : 添加默认音频、视频转场
- `ctrl+shift+D` : 添加默认音频转场


# 9. 其他

- `M` : 添加标记，注释，可以在标记面板中查看所有标记
- 一般直接为时间轴添加标记而不为片段单独添加标记
- 两次 `MM` 可以添加并编辑标记
