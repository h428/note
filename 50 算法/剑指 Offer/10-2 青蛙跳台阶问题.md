

# 1. 题意

- 一只青蛙一次可以跳上1级台阶，也可以跳上2级台阶。
- 求该青蛙跳上一个 n 级的台阶总共有多少种跳法


# 2. 动态规划，状态转移方程是一个斐波那契数列

```java
class Solution {
    public int numWays(int n) {

        // dp[i] : 跳到第 i 级台阶的跳法，则
        // dp[1] = 1
        // dp[2] = 2, 跳两次 1 或者跳一次 2
        // dp[i] = dp[i-2] + dp[i-1]
        // 第 i 级台阶可从 i-2 台阶跳到或者由 i-1 台阶跳到，因此可以利用子状态
        // 可以看出，dp[i] 是个斐波那契数列，因此可以利用两个变量迭代优化空间

        int a = 1, b = 1, tmp;

        for (int i = 2; i <= n; ++i) {
            tmp = b;
            b = (a + b) % 1000000007;
            a = tmp;
        }

        return b;
    }
}
```
